# 🔍 IP Logger Tool

A simple and powerful IP Logger tool made for educational and security awareness purposes.  
This project helps you understand how IP tracking works and how connections are established over the internet.

---

## ⚙️ Features
- Easy to use
- Fast IP logging
- Lightweight
- Clean output

---

## 🚀 Usage

```bash
git clone https://github.com/Cyber75480/Ip-logger.git
cd Ip-logger
chmod +x run.sh
bash run.sh
```
